Learning Human Actions

We intend to build an agent which could identify and classify various human actions that occur in a video (Eg: movies, TV series, news etc.). This would include these 3 important tasks:

  • 1. Collecting clips belonging to each activity of interest to train the agent. Here is a dataset of such actions collected automatically using movie scripts.
  • 2. Video representation using Space-Time Interest Points (STIP).
  • 3. Classification of actions using Non-linear support vector machine with multi channel kernel.

Reference - [1]

We also intend to implement joint learning [2] which combines knowledge from different activities to get to a conclusion. For instance, if the agent has been trained to detect “getting out of a car” and “walk by a building”, it should be able to detect “getting out of a building“. This would be benificial in surveillance for security reasons etc. If joint learning is implemented, the agent would require very minimal training.

Disclaimer - The approach to the problem and whether or not to implement joint learning is subject to change depending on the difficulty of the task and availability of time.

References:

[1] Ivan Laptev, Marcin Marszalek, Cordelia Schmid, Benjamin Rozenfeld. Learning realistic human actions from movies. IEEE Conference on Computer Vision & Pattern Recognition, 2008.
[2] Gang Wang, David Forsyth. Joint learning of visual attributes, object classes and visual saliency. IEEE 12th International Conference on Computer Vision (ICCV), 2009.
